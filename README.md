# fsuseradd
add extension/user file for freeswitch

## Usage
```
Usage:
  fsuseradd

Application Options:
  -s, --start=   start number
  -e, --end=     end number
  -o, --output=  output dir
  -h, --help     show this help message
  -r, --remove   remove files
  -v, --version  show version message


fsuseradd -s 1020 -e 1100

```
